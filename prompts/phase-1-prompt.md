# MiraPrompt Phase 1 Prompt

Build the Phase 1 UI only.

Requirements:
1. Quasar SPA frontend in miraprompt-ui.
2. Load local style JSON data from frontend assets.
3. Category dropdown and dependent subcategory dropdown.
4. Merge style options using union from Global + category + subcategory.
5. Render style dimensions as expansion items.
6. Render grouped options with alternating group backgrounds.
7. Multi-select responsive checkboxes for options.
8. Button opens modal showing selected options only in JSON.
9. No persistence API or image generation yet.
